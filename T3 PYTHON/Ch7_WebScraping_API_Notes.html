Python Course Notes

1. List of Imports Used in the Files

  • import requests - Used to send HTTP requests to websites or APIs.
  • from bs4 import BeautifulSoup - Used to parse HTML and extract data.
  • import pandas as pd - Used to structure extracted data into tables/DataFrames.
  • import pprint - Used to neatly print dictionaries and JSON data.
  • import json - Used to parse and save JSON data.
  • import datetime - Used to convert timestamps to readable dates.

2. How to Read from a Link (Website)

Using requests.get() to fetch a live webpage and BeautifulSoup to parse it:

response = requests.get(r'https://subslikescript.com/movie/Batman_The_Enemy_Within-7170214')
content = response.text
data = BeautifulSoup(content, 'html.parser')

print(data.prettify()) # To format the HTML nicely

4. How to Find Elements in the HTML

Using find() for a single element and find_all() for multiple elements:

# Finding a single element (e.g., h1 tag)
title = data.find('h1')
print(title.text)

# Finding multiple elements (e.g., all divs with a specific class)
scripts = data.find_all("div", class_="full-script")

# Extracting links (href attribute)
links = data.find_all('a')
for link in links:
    print(link['href'])

5. How to Store Scraped Data into a Pandas DataFrame

Iterate through elements, save them to a dictionary of lists, and convert to a DataFrame:

# Extracting elements
jobs = data.find_all("div", class_="job_seen_beacon")

# Setting up a dictionary
job_dict = {'Title': [], 'Company': [], 'Location': []}

for i in jobs:
    t = i.find("h2", class_="jobTitle")
    job_dict["Title"].append(t.text)
    
    c = i.find("div", class_="css-1afmp4o")
    job_dict["Company"].append(c.text)

# Creating the DataFrame
df = pd.DataFrame(job_dict)
df

6. How to Read from JSON REST APIs

Using requests to fetch JSON data, passing parameters, and parsing the response:

api_Key = "6418d6e58a08b69ee8b6c947e98e3c3a"
url = f"http://api.openweathermap.org/geo/1.0/direct"

# Passing query params
params = {'q': 'Ahmedabad', 'limit': 5, 'appid': api_Key}
response = requests.get(url, params=params)

# Reading the JSON response
result = response.json()
pprint.pprint(result)

7. How to Write and Read Local Files

Writing to a Text File

title = "Movie Name"
script = "Script content here..."

f = open(title + '.txt', 'w', encoding='utf-8')
f.write(script)
f.close()

Writing and Reading JSON Files

import json

# Writing to a JSON file
with open('demo.json', 'w') as f:
    json.dump(result, f, indent=4)

# Reading from a JSON file
with open('demo.json', 'r') as f:
    data = json.load(f)
print(data)
